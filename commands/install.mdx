title: "install"
description: "Download and install a specific version of Xcode."
---

## Synopsis

```bash
xcodes install [<version>] [options]
```

Download and install a specific version of Xcode. By default, xcodes uses a `URLSession` to download the specified version. If [aria2](https://aria2.github.io) is installed (available via Homebrew), it will be used automatically for faster downloads using up to 16 parallel connections.

<Warning>
  Installation requires superuser (root) privileges for certain steps, such as granting Simulator runtime permissions and running `xcode-select`. xcodes will prompt you for your password when needed. Use `--no-superuser` to skip these steps.
</Warning>

## Options

| Flag | Description |
|------|-------------|
| `<version>` | The Xcode version to install. Accepts release versions (`16.2`), betas (`11 Beta 7`), and GM seeds (`11.2 GM seed`). |
| `--path <path>` | Local path to an Xcode `.xip` file. Skips downloading. |
| `--latest` | Update the list and install the latest release version. |
| `--latest-prerelease` | Update the list and install the latest prerelease version, including GM seeds and GMs. |
| `--aria2 <path>` | Path to an `aria2` executable. Searches `$PATH` by default. |
| `--no-aria2` | Disable `aria2` even if it is available. |
| `--select` | Select the installed version as the active Xcode after installation completes. |
| `-p`, `--print-path` | Print the path of the currently selected Xcode. Used in combination with `--select`. |
| `--update` | Refresh the list of available versions before installing. |
| `--experimental-unxip` | Use experimental unxip for unarchiving. Can be up to 2–3x faster. |
| `--no-superuser` | Skip installation steps that require superuser privileges. |
| `--empty-trash` | Permanently delete the `.xip` archive after installation instead of moving it to the Trash. |
| `--directory <path>` | Directory to install Xcode into. Defaults to `/Applications`. Also reads the `XCODES_DIRECTORY` environment variable. |
| `--use-fastlane-auth` | Authenticate using a fastlane spaceship session. |
| `--fastlane-user <user>` | The fastlane spaceship username to use. |
| `--data-source <source>` | Source for available Xcode versions. One of `apple` or `xcodeReleases`. Defaults to `xcodeReleases`. |
| `--color` / `--no-color` | Enable or disable colored output. |

<Note>
  If `aria2` is installed and available in `$PATH`, xcodes uses it automatically to download Xcode up to 3–5x faster using up to 16 parallel connections. Install it with `brew install aria2`. To opt out, pass `--no-aria2`.
</Note>

## Examples

<CodeGroup>

```bash Install a release version
xcodes install 16.2
```

```bash Install a beta
xcodes install "11 Beta 7"
```

```bash Install a GM seed
xcodes install "11.2 GM seed"
```

```bash Install and select immediately
xcodes install 16.2 --select
```

```bash Install the latest release
xcodes install --latest
```

```bash Install the latest prerelease
xcodes install --latest-prerelease
```

```bash Install from a local .xip file
xcodes install 16.2 --path ~/Downloads/Xcode_16.2.xip
```

```bash Install to a custom directory
xcodes install --latest --directory "/Volumes/External/"
```

```bash Install with faster unarchiving
xcodes install --latest --experimental-unxip
```

</CodeGroup>
