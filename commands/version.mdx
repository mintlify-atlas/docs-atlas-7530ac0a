---
title: "version"
description: "Print the version number of xcodes itself."
---

## Synopsis

```bash
xcodes version [options]
```

Print the version number of the installed xcodes binary.

## Options

| Flag | Description |
|------|-------------|
| `--color` / `--no-color` | Enable or disable colored output. |

## Example

```bash
xcodes version
```

Sample output:

```
1.4.0
```

<Tip>
  Use this command to confirm which version of xcodes is installed, especially when troubleshooting or filing a bug report.
</Tip>
