## What xcodes does

<CardGroup cols={2}>
  <Card title="Install any Xcode version" icon="box-archive">
    Install stable releases, betas, GM seeds, and release candidates by version number.
  </Card>
  <Card title="Switch instantly" icon="arrows-rotate">
    Use `xcodes select` to switch the active Xcode globally, or read from a `.xcode-version` file.
  </Card>
  <Card title="Fast downloads" icon="bolt">
    Uses aria2 with up to 16 parallel connections for downloads 3–5x faster than a browser.
  </Card>
  <Card title="Simulator runtimes" icon="mobile">
    Download and install iOS, watchOS, tvOS, and visionOS simulator runtimes alongside Xcode.
  </Card>
</CardGroup>

## Get started in 30 seconds

<Steps>
  <Step title="Install xcodes">
    ```bash
    brew install xcodesorg/made/xcodes
    ```
  </Step>
  <Step title="List available Xcode versions">
    ```bash
    xcodes list
    ```
  </Step>
  <Step title="Install the version you need">
    ```bash
    xcodes install 16.2
    ```
    You'll be prompted for your Apple ID on first run. Credentials are stored securely in your keychain.
  </Step>
  <Step title="Switch between versions">
    ```bash
    xcodes select 16.2
    ```
  </Step>
</Steps>

<Tip>
  Add a `.xcode-version` file to your repository so that every team member and CI machine uses the same Xcode version automatically. See the [`.xcode-version` guide](/guides/xcode-version-file) for details.
</Tip>
